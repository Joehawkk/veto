# VETO — Осознанные покупки

Мобильное веб-приложение для студентов, которое помогает принимать взвешенные решения перед покупками. 4 быстрых вопроса → честный ИИ-вердикт.

---

## Как это работает

1. Вводишь товар и цену
2. Отвечаешь на 4 вопроса: нужно сейчас? есть похожее? как долго думал? какое настроение?
3. ИИ выносит вердикт: **Купи / Подожди / Не покупай**
4. Для импульсных покупок (бургер, кофе, доставка) — предлагает конкретные альтернативы
5. Можно поставить таймер "подумать" на 1–7 дней
6. Отказался — сумма копится в историю

---

## Функции

- **ИИ-вердикт** — Claude Haiku 4.5 через OpenRouter с серверными override-правилами
- **Умные категории** — лекарства и базовые продукты всегда "купи"; кофе/фастфуд никогда "купи"
- **Аналоги для еды** — при импульсной еде ИИ предлагает приготовить дома или купить дешевле
- **Ценовой анализ** — ИИ замечает подозрительно низкие цены (риск подделки)
- **Таймер решения** — отложи на 1–7 дней, получи напоминание
- **История** — все проверки, статистика сбережений, детальные карточки
- **Группы** — делись отказами с друзьями, получай респект!
- **Цели** — копи на мечту, прогресс-бар считает отказы в деньгах
- **Онбординг** — цели, триггеры трат, интересы, бюджет

## Стек

### Frontend
- React 18 + TypeScript + Vite
- Tailwind CSS
- React Router v6

### Backend
- Go + Fiber v2
- PostgreSQL
- JWT-авторизация

### AI
- [OpenRouter](https://openrouter.ai) → Claude Haiku 4.5 (`anthropic/claude-haiku-4-5`)
- Серверные override-правила (не доверяем модели для критичных категорий)
- Fallback: локальный скоринг без API

## Запуск

### Требования
- Node.js 18+
- Go 1.21+
- PostgreSQL 14+

### Backend

```bash
cd backend
cp .env.example .env   # заполни переменные
go run main.go
```

`.env`:
```env
DATABASE_URL=postgres://veto:veto@localhost:5432/veto?sslmode=disable
JWT_SECRET=your-secret-32-chars
PORT=8080
OPENROUTER_API_KEY=sk-or-v1-...
OPENROUTER_MODEL=anthropic/claude-haiku-4-5
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Приложение: `http://localhost:5173`

### Публичный доступ (Cloudflare Tunnel)

```bat
start.bat   # запускает всё + выдаёт публичный URL
stop.bat    # останавливает
```

## ИИ-логика

Вердикты определяются в два этапа:

**1. Локальный скоринг** (`backend/handlers/ai.go → deriveAIVerdict`)
- Базовый счёт: −1 (скептично по умолчанию)
- Импульсный товар (кофе/бургер/доставка): −2
- Нужно сейчас: +2 / не нужно: −2
- Есть похожее: −2 / нет: +1
- Думал 3+ дня: +3 / 24ч: +1 / 1ч: −1 / 30 мин: −2
- Плохое настроение: −2
- Ночное время (21:00–06:00): −1

**2. Claude Haiku 4.5** — выводит вердикт советом, НО:
- Для лекарств/базовых продуктов/гигиены → всегда "go" (серверный override)
- Для импульсных товаров → никогда "go"; дороже 400 ₽ → "veto"
- ИИ не может быть строже, чем локальный скоринг (cap)
