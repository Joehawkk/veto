# VETO — Осознанные покупки для студентов

> Умный AI-помощник, который помогает не тратить деньги на эмоциях

---

## Что такое VETO?

VETO — это персональный инструмент проверки покупок. Перед тем как что-то купить, ты отвечаешь на 4 вопроса, и AI выносит вердикт: **купи**, **подожди** или **не покупай**. Приложение учитывает твоё настроение, время суток, историю покупок и личный профиль.

---

## Функционал

### Онбординг (5 экранов)
- **Цель** — зачем ты здесь: экономить, не тратить на эмоциях, копить на мечту и т.д.
- **Триггеры трат** — что провоцирует импульсные покупки: стресс, усталость, скидки, реклама
- **Интересы** — до 4 тем (игры, музыка, мода, книги и др.) — используются для альтернативных советов
- **Ежемесячные траты** — слайдер 0–50 000 ₽
- **Цель накоплений** — опциональная сумма и срок

### Проверка покупки
Форма ввода: название товара, цена, наличие скидки.

Затем 4 вопроса:
1. **Нужно прямо сейчас?** — потребность vs желание
2. **Есть похожее?** — есть ли аналог дома
3. **Как долго думал?** — 30 мин / 1 час / 24 часа / 3+ дня
4. **Настроение** — 6 вариантов (хорошее, нейтральное, грустное, злое, стрессовое, усталое)

### AI-вердикт
- Три варианта: `go` (купи), `wait` (подожди), `veto` (не покупай)
- AI от OpenRouter с fallback-цепочкой из 3 моделей
- Учитывает время суток (ночные покупки = высокий риск), настроение, историю
- Отображается как советник-друг, без морализаторства

### Экран результата
- Цветной блок с вердиктом и AI-советом
- Таймер "подумать": 1 день / 2 дня / 3 дня / неделю
- Кнопки: **Отказываюсь** или **Куплю**
- После отказа — экран **"Так держать!"** с суммой сбережений и советом по интересам

### История
- Список всех проверок с датой, ценой, вердиктом, итогом
- Нажми на товар → **детальная карточка** с ответами, AI-комментарием, таймером
- Для ожидающих товаров: кнопки «Отказался» / «Купил» прямо в карточке
- Статистика: сумма сбережений, количество отказов, куплено

## Технологии

### Frontend
| Технология | Версия |
|---|---|
| React | 18 |
| TypeScript | 5 |
| Vite | 5 |
| Tailwind CSS | 3 |
| React Router | 6 |
| uuid | 9 |

### AI
- **API**: [OpenRouter](https://openrouter.ai)
- **Модели (fallback chain)**:
  1. `google/gemma-4-26b-a4b-it:free`
  2. `openai/gpt-oss-120b:free`
  3. `openai/gpt-oss-20b:free`

### Хранилище
- **localStorage** — профиль, история, флаг онбординга
- **sessionStorage** — текущий товар и результат проверки
- Без бэкенда, без авторизации

## Запуск

```bash
cd frontend
npm install
cp .env.example .env   # добавь VITE_OPENROUTER_API_KEY
npm run dev
```

Приложение доступно на `http://localhost:5173`

### Переменные окружения

```env
VITE_OPENROUTER_API_KEY=sk-or-v1-...
```

## Структура проекта

```

├── src/
│   ├── pages/
│   │   ├── Home.tsx           # Главная + форма ввода товара

│   ├── lib/
│   │   ├── storage.ts         # Все операции с localStorage
│   │   └── scoring.ts         # Локальный скоринг + метаданные вердиктов
│   └── App.tsx                # Роутинг
```
